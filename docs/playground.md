## Ideia: App de Controle de Gastos Pessoais

### Problema
- Falta de visibilidade sobre para onde o dinheiro vai
- Dificuldade de manter orçamento por categoria
- Sem controle de gastos recorrentes (assinaturas, aluguel)

### Público
- Uso pessoal (single user)

### Funcionalidades MVP

**Cadastro e gestão:**
- Cadastrar gastos (valor, categoria, data, descrição)
- Cadastrar receitas (entradas de dinheiro)
- Gastos recorrentes (aluguel, assinaturas — lançamento automático mensal)
- Categorias personalizáveis

**Visualização:**
- Dashboard com resumo mensal (total gasto, total recebido, saldo)
- Gráficos (pizza por categoria, barras por mês)
- Filtros por período (semana, mês, intervalo customizado)

**Orçamento:**
- Definir limite de gasto por categoria
- Alerta visual quando próximo/acima do limite

**Exportação:**
- Exportar dados para CSV

**Mobile:**
- PWA (instalar no celular como app, funcionar offline)

### O que NÃO é MVP
- Multi-usuário / compartilhamento
- Integração com bancos (Open Finance)
- Importação de extrato bancário
- Metas de economia
- Contas a pagar/receber com vencimento
